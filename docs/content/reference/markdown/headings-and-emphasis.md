Heading are indicated with a hash.

# Heading 1

```md
# Heading 1
```

## Heading 2

```md
## Heading 2
```

### Heading 3
```md
### Heading 3
```

Use either bold, italics, or monospaced font styling for emphasis:

*Italicized text*
```md
*Italicized text*
```

**Bold text**
```md
**Bold text**
```

`Identifiers and code`
```md
`Identifiers and code`
```
